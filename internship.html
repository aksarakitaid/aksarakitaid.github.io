Sekilas tentang Kampus Merdeka

PT. Jalin Mayantara Indonesia

Mengacu kepada program-program di lingkungan pendidikan tinggi, serta dalam upaya memberikan pengalaman kepada mahasiswa di perguruan tinggi sebagai aset generasi masa depan bangsa. PT. Jalin Mayantara Indonesia bermaksud untuk membuka peluang kerjasama dengan pendidikan tinggi dengan memberikan kesempatan kepada mahasiswa untuk dapat berpartisipasi secara aktif terhadap dunia industri teknologi informasi melalui program magang merdeka.

Why Internship?

Key Point

Kriteria Internship

  1. Peserta Merupakan Mahasiswa Aktif
  2. Bersedia Magang selama 6 Bulan
  3. Minimal S1 : Semester 6

Keuntungan Internship

  1. Mendapatkan Sertifikat Industri dan atau Kompetensi
  2. Uang Saku sesuai Aturan Berlaku
  3. Pengganti mata kuliah yang relevan

Bidang Keahlian

UXD
UX Designer
Tersedia
  • Memahami metode design thinking
  • Memahami pembuatan user flow dan mockup aplikasi menggunakan adobe xd, figma, atau yang lainnya
BEE
Backend Engineer
Tersedia
  • Memahami dasar bahasa pemrograman PHP
  • Menerapkan salah satu PHP MVC framework seperti Laravel, Symfony, Code Igniter atau YII
  • Memahami pengaksesan dan pengelolaan data dalam RDBMS (Mysql, MariaDB)
  • Diutamakan memahami penerapan RESTful web services
FEE
Frontend Engineer
Tersedia
  • Memahami layout halaman web menggunakan HTML 5 dan CSS 3
  • Memahami dasar Javascript, ES6+ dan implementasi AJAX Request dalam sebuah aplikasi web
  • Menerapkan salah satu satu framework Javascript modern, seperti Vue, Angular, React, Backbone atau Ember
  • Memahami implementasi Restful API
AND
Android Developer
Tersedia
  • Memahami bahasa pemrograman Java dan Kotlin
  • Memahami komunikasi data menggunakan REST dan JSON
  • Menguasai Android Jetpack lebih diutamakan
DTE
Data Engineer
Tersedia
  • Memahami teknik dasar kueri data dengan SQL
  • Memahami konsep dasar infrastruktur jaringan komputer dan administrasi sistem basis data
  • Memahami tahapan data warehouse yang mencakup identifikasi sumber data, ETL, dan ekosistem data warehouse
  • Memahami pendekatan integrasi data antar sistem basis data maupun pertukaran data melalui antarmuka aplikasi
  • Mengenal arsitektur basis data relasional, data warehouse, serta ekosistem big data
